# Changelog

All notable changes to the Zappy project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-23

### Added
- Complete multiplayer game server (C, poll-based networking)
- 3D graphical client (C++17, Raylib, CMake)
- AI player client (Python, TCP)
- Full protocol support for AI commands (forward, left, right, look, inventory, take, drop, broadcast, fork, incantation)
- Server-side game mechanics: food consumption, level progression, team-based incantations
- GUI features: real-time player tracking, tile inspection, broadcast history, sound effects
- AI strategy algorithms: leader election, resource gathering, team coordination
- Comprehensive test suite for core server functionality
- Documentation for architecture, API, and deployment

### Initial Release
- Project submitted for Epitech evaluation
- All core requirements implemented and tested
